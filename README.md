# Flowchart OpenClaw

A lightweight Mermaid-based flowchart app for quickly visualizing process flows.

## Features

- Mermaid text editor (flowchart / mindmap / sequence, etc.)
- Live preview
- Export as SVG
- Export as PNG
- Auto-saves latest diagram in localStorage

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown by Vite.

## Build

```bash
npm run build
npm run preview
```

## Notes

- Best for fast iteration in Replit or local browser.
- This app intentionally keeps scope simple so it can be extended quickly.
